Desligue o computador e verifique fisicamente se a ventoinha (cooler) está a girar. Retire o pó acumulado, certifique-se de que o cabo da ventoinha está bem ligado à placa-mãe (no pino "SYS_FAN" ou "CPU_FAN"). Se a ventoinha estiver avariada, terá de ser substituída para evitar sobreaquecimento.

Se a ventoinha estiver a girar normalmente e o erro continuar a aparecer, a mensagem pode ser um falso positivo ou indicar que o sensor de velocidade da ventoinha falhou. Neste caso, pode contornar a situação ajustando a BIOS/UEFI:Reinicie o computador e prima repetidamente a tecla F2 (ou F10, dependendo do seu modelo) para aceder à BIOS.Procure por separadores como Advanced, Power ou Hardware Monitor.Procure opções relacionadas com Fan Control ou Q-Fan Configuration.Encontre o limite inferior da velocidade da ventoinha ou a monitorização da ventoinha do sistema (System Fan) e altere a definição para Ignore ou Disabled.Guarde as alterações e reinicie o computador (normalmente ao premir F10 para salvar e sair).Aviso: Desativar o alerta de falha de ventoinha na BIOS é apenas uma solução temporária para eliminar o erro se souber que a ventoinha está operacional. Utilizar o PC com uma ventoinha realmente avariada pode danificar o processador e outros componentes devido ao sobreaquecimento.

A forma mais rápida de iniciar um servidor local é usar o servidor embutido do PHP. 
Abra o terminal (ou Prompt de Comando), navegue até a pasta dos seus arquivos usando cd, e execute:
    php -S localhost:8000. 
Em seguida, acesse http://localhost:8000 no seu navegador.
Usando o Servidor Embutido (Ideal para testes rápidos).

Para iniciar rapidamente um servidor web sem precisar instalar programas pesados como o Apache ou o Nginx, o próprio PHP possui um servidor nativo.
Abra o terminal (Terminal no Mac/Linux ou Prompt de Comando/PowerShell no Windows).
Navegue até o diretório onde estão seus arquivos PHP, utilizando o comando cd seguido do caminho da pasta:
bash:
    cd caminho/para/sua/pasta
Use o código com cuidado.Inicie o servidor com o comando abaixo:
bash:
    php -S localhost:8000
Use o código com cuidado.Nota: O número 8000 é a porta onde o servidor vai rodar, mas você pode alterá-lo se necessário.
Visualize no navegador: Abra o seu browser de preferência e acesse o endereço:http://localhost:8000
